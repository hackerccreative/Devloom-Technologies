## Getting Started

```bash
# Clone the repo
git clone https://github.com/your-username/devloom-technologies.git
cd devloom-technologies

# Install dependencies
npm install

# Start dev server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.
